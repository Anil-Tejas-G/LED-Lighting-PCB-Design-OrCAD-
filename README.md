# LED Lighting PCB Design (OrCAD)

## Overview

This project involves the design and implementation of a multi-LED control system using digital logic components and PCB design tools.

## Features

* LED control using shift register (74HC164)
* Logic-based signal processing using gates (74HC14, 74HC86)
* Timing circuit for controlled LED sequencing
* Stable power regulation using voltage regulator components

## Design Workflow

1. Developed circuit schematic using OrCAD
2. Selected components including logic ICs, resistors, capacitors, and LEDs
3. Converted schematic into PCB layout
4. Verified routing and component placement
5. Analysed signal flow and output behaviour

## Tools Used

* OrCAD (schematic & PCB design)
* PSpice (circuit simulation)

## Key Learning Outcomes

* Understanding of digital logic circuits and signal propagation
* PCB design workflow from schematic to layout
* Integration of logic circuits with real hardware implementation
* Practical experience with component-level design and debugging
